IRC-BOT
=======
My TCL coded IRC bot!

Owner: Cajs (Cameron Stephen)

## Prerequisites 
From Debian / Ubuntu:  
    $ sudo apt-get install tcl tcl-dev tcl8.6 tcl8.6-dev  
For Red Hat:  
    $ sudo yum install tcl tcl-devel
